# Homebrew Distribution Setup

This document explains how to set up Homebrew distribution for Terminal Agent.

## Overview

Terminal Agent uses [GoReleaser](https://goreleaser.com/) to automate the release process, including:
- Building binaries for multiple platforms (Linux/macOS, amd64/arm64)
- Creating GitHub releases with changelogs
- Automatically updating the Homebrew tap formula

## Prerequisites

### 1. Create a Homebrew Tap Repository

Create a new GitHub repository named `homebrew-tap` under the `laszukdawid` account:

1. Go to https://github.com/new
2. Name the repository: `homebrew-tap`
3. Make it public
4. Initialize with a README

The repository structure should be:
```
homebrew-tap/
├── README.md
└── Formula/
    └── terminal-agent.rb  (auto-generated by GoReleaser)
```

### 2. Create GitHub Personal Access Token

Create a Personal Access Token (PAT) with `repo` scope:

1. Go to https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Name: `HOMEBREW_TAP_GITHUB_TOKEN`
4. Select scope: `repo` (Full control of private repositories)
5. Generate and copy the token

### 3. Add Secret to Terminal Agent Repository

Add the token as a repository secret:

1. Go to https://github.com/laszukdawid/terminal-agent/settings/secrets/actions
2. Click "New repository secret"
3. Name: `HOMEBREW_TAP_GITHUB_TOKEN`
4. Value: (paste the token from step 2)

## How It Works

### Release Flow

1. When a new tag `v*.*.*` is pushed to the main repository
2. GitHub Actions triggers the release workflow
3. GoReleaser:
   - Builds binaries for all platforms
   - Creates tar.gz archives
   - Generates checksums
   - Creates a GitHub release
   - Updates the formula in `laszukdawid/homebrew-tap`

### Formula Auto-Update

GoReleaser automatically:
- Calculates SHA256 checksums for each archive
- Updates the formula with new version and checksums
- Commits and pushes to the tap repository

## Manual Testing

### Test GoReleaser Locally

```sh
# Install GoReleaser
brew install goreleaser

# Test the configuration (dry run)
goreleaser check

# Build a snapshot (no release)
goreleaser build --snapshot --clean

# Full release dry run
goreleaser release --snapshot --clean
```

### Test Homebrew Installation

After a release, test the installation:

```sh
# Add the tap
brew tap laszukdawid/tap

# Install
brew install terminal-agent

# Verify
agent --help

# Upgrade (after new release)
brew upgrade terminal-agent
```

## Troubleshooting

### Formula Not Updating

1. Check the GitHub Actions logs for errors
2. Verify `HOMEBREW_TAP_GITHUB_TOKEN` secret is set correctly
3. Ensure the token has `repo` scope
4. Check if the tap repository exists and is accessible

### Installation Fails

1. Check the formula syntax: `brew audit --strict terminal-agent`
2. Verify the download URLs are accessible
3. Check SHA256 checksums match

### Build Errors

1. Run `goreleaser check` to validate configuration
2. Ensure Go version matches (1.23+)
3. Check for CGO dependencies (should be disabled)

## Alternative: Homebrew Core

For wider distribution, you can submit to Homebrew Core:

**Requirements:**
- Project must be notable (500+ GitHub stars recommended)
- Stable release history
- No vendored dependencies issues
- Follows Homebrew formula guidelines

**Process:**
1. Fork https://github.com/Homebrew/homebrew-core
2. Create formula in `Formula/t/terminal-agent.rb`
3. Submit pull request
4. Address reviewer feedback

For most projects, a personal tap is sufficient and easier to maintain.
